blog
===

my blog - because "EVERYBODY needs one" not

## Prerequisites

- `npm i -g pecoup`
- `npm i`

## Development

Extract from sample hello-world post ;)  

```bash
# Run dev server
npm run dev

# Build to .peco/website directory
# Which can be deployed as a static website
npm run build
```

## Usage

### add post

`pecoup new post [post title]`

### add page

`pecoup new page [page title]`
